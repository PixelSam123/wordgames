# wordgames

A high performance word games server for WebSockets

Word bank retrieved from Datamuse: https://www.datamuse.com/api/

---

## Spinning it up

Simply run the binary, or `cargo run`

## List of games

1. `ws/anagram` Normal anagrams game. Plans:
